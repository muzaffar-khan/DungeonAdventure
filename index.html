Description:

A text-based adventure game is a completely text-based and very simple game.
In this game we have to play against the enemy. The enemy will be generated randomly. We have options to play the game. We can attack the enemy by pressing 1 or we can use health potion to increase our health or we can run from the current enemy and play with another enemy.
If we’ll win the game, we have 2 options, we can either continue playing the game with other enemies or we can exit the game. If the player lost the game against enemy the game gets exited with the message that the player is weak to go.

Algorithm:

1. Start
2. Create a class for DungeonAdventure.
3. Create enemies with string and declare it’s max health and enemy attack damages.
4. Declare players health, attack damage, number of health potion, etc.
5. Take name of the user using scanner.
6. Create a while loop in a while loop with all the conditional statements to be used.
7. Break the loop if players health is less than 1 and declare that the player is weak to go.
8. If the player wins create a while loop for choice whether the player has to continue playing or they want to exit the game.
9. Stop

Working:

In this game we have to play against the enemy. First, it’ll ask to enter our name once we entered the name and pressed enter the enemy is generated randomly. String is used for enemies in which 4 enemies are there namely “Skeleton", "Zombie", "Warrior", "Assassin". We have options to play the game. We can attack the enemy by pressing 1 or we can use health potion to increase our health by pressing 2 or we can run from the current enemy and play with another enemy by pressing 3. If we’ll win the game against the enemy, we have 2 options, we can either continue playing the game with other enemies by pressing 1 or we can exit the game by pressing 2.
